# Jump_Frog



### Key
* left/right: Frog jump angle          (0 - 180)
* up/down: Frog jump magnitude         (0 - 0.25)
* f: Toggle filled/wireframe rendering
* g: Toggle animation on/off
* o: Toggle OSD on/off
* r: Respawn
* +(=)/-: Double/halve wave tesselation
* ' ' (space): From jump
* q/Esc: Quit the program
